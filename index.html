When the agent drafts the workpaper, Optro is the evidence-of-truth backend.

If Anthropic ships SOX-aware agents and YC speed-shops follow, the Controller stops opening the Optro UI — they ask Claude Desktop questions and expect audit-ready output back. Hallucination is the existential risk and the structured citation envelope is the moat. The MCP server forces the agent to surface optro_record_id citations on every claim and to flag REQUIRES HUMAN REVIEW wherever judgment was applied. Anyone can wrap an LLM around a chat box; what stops the management assertion from being fabricated is the evidence-of-truth backend returning exactly the citations the agent is allowed to use.

The scenario thesis
Foundation models ship SOX-aware agents directly.
Claude / ChatGPT become the workpaper-drafting interface. Vertical players survive only with deep data + workflow integration that the model needs. The product surface moves from the human's screen to the model's tool-call envelope.
What Provenance hedges
The structured citation envelope around every tool response.
Five MCP tools, every response wrapped with optro_record_id citations, schema-versioned, sha256-verified at ingest. The agent cannot fabricate a defensibility score because the MCP contract forces the citation on the wire.
Why no other vendor can do this
The data shape is the moat.
An LLM wrapper that doesn't sit on the source-of-record systems can ask the agent to cite — but cannot guarantee the citation is real. Provenance's MCP returns only artifacts that exist in NetSuite, Salesforce, Workday at ingest time. The citation is structurally enforced, not stylistically requested.
Trigger condition · ship the MCP server commercially when this fires
Anthropic ships a SOX agent AND 2+ funded LLM-wrapper startups raise $20M+. The repo's server.ts + provenance.ts are real, runnable code today — Phase-1 is hardening the auth + tenant model, not building the surface. The pre-positioning lives in Context; Provenance is the evidence-of-truth layer Context calls.

When software ARR breaks, the FDE pod is what we sell.

If Controllers use Claude directly and renewal economics collapse, Optro pivots margin to the Practice service line. The defensible value becomes the production line itself — agent drafts the routine 76% in seconds, FDE pod owns the judgment-shaped 24%, the audit trail captures the seam, and the external auditor signs the deliverable without rework. Measured at 4% rework rate this quarter vs the 18% industry baseline for boutique advisory firms. Foundation models cannot replace that workflow without replacing the FDE pod, and the FDE pod is the moat the production tool is designed to scale.

The scenario thesis
Software ARR economics break.
Controllers use Claude / ChatGPT directly. Vertical SaaS becomes commodity. Renewals fall, multiples compress. The only defensible position is the human-plus-machine service delivery that audit firms actually accept on first review.
What Provenance hedges
The FDE production line itself.
Three pod members deliver what used to take a 12-person Big-4 senior team. Agent + scorer save 1,840 hours per quarter. The audit trail captures every agent draft, every FDE edit, every customer sign-off — so the partner across the table can verify the work without re-doing it.
The KPI that proves the moat
4% rework vs 18% industry baseline.
14 workpapers shipped this quarter; 3 triggered any auditor follow-up; 0 required substantive re-test. Boutique advisory firms typically see 15-22% rework on first auditor review. That 14-point spread is the production-line product.
Trigger condition · pivot revenue to services when this fires
