Scenario thesis

Group B · Options 4–6

When the agent drafts the workpaper, Optro is the evidence-of-truth backend.

If foundation models ship SOX-aware agents and Controllers stop opening the Optro UI, hallucination is the existential risk. The structured citation envelope around every tool response is the moat. The MCP server forces the agent to surface optro_record_id citations on every claim and to flag REQUIRES HUMAN REVIEW wherever judgment was applied.

Trigger Ship the MCP server commercially when Anthropic ships a SOX agent AND 2+ funded LLM-wrapper startups raise $20M+. The repo's server.ts + provenance.ts are real, runnable code today. Pre-positioning lives in Context.
Why this hedge matters
Scenario thesis
Foundation models ship SOX-aware agents directly.
Claude / ChatGPT become the workpaper-drafting interface. Vertical players survive only with deep data + workflow integration the model needs. The product surface moves from the human's screen to the model's tool-call envelope.
What Provenance hedges
The structured citation envelope on every tool response.
Five MCP tools wrapped with optro_record_id citations, schema-versioned, sha256-verified at ingest. The agent cannot fabricate a defensibility score because the MCP contract forces the citation on the wire.
Why no one else can do this
The data shape is the moat.
An LLM wrapper not sitting on the source-of-record can ask the agent to cite — but cannot guarantee the citation is real. Provenance's MCP returns only artifacts that exist in NetSuite, Salesforce, Workday at ingest time. The citation is structurally enforced, not stylistically requested.

Why this hedge matters
Scenario thesis
Software ARR economics break.
Controllers use Claude directly. Vertical SaaS becomes commodity. Renewals fall, multiples compress. The only defensible position is the human-plus-machine service delivery that audit firms accept on first review.
What Provenance hedges
The FDE production line itself.
Three pod members deliver what used to take a 12-person Big-4 senior team. Agent + scorer save 1,840 hours per quarter. The audit trail captures every agent draft and FDE edit so the partner can verify without re-doing.
The KPI that proves the moat
4% rework vs 18% industry baseline.
14 workpapers shipped this quarter; 3 triggered any auditor follow-up; 0 required substantive re-test. Boutique advisory firms typically see 15–22%. That 14-point spread is the product.
